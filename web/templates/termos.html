1. Aceitação

Ao utilizar a plataforma Prazu ("Serviço"), você concorda com estes Termos de Uso. Caso não concorde, não utilize o Serviço.

2. Descrição do Serviço

A Prazu oferece monitoramento automatizado de prazos processuais, consulta ao Diário de Justiça Eletrônico Nacional (DJEN) e envio de notificações via WhatsApp. O Serviço é uma ferramenta de apoio e não substitui o acompanhamento profissional dos processos.

3. Cadastro

Para utilizar o Serviço, é necessário criar uma conta com informações verdadeiras. Você é responsável por manter a confidencialidade de suas credenciais de acesso.

4. Período de Teste

Novos usuários recebem um período gratuito de teste de 7 (sete) dias. Após o término, o acesso será limitado até a contratação de um plano.

5. Responsabilidades do Usuário

  • Fornecer dados corretos de OAB e contato
  • Verificar os prazos informados antes de tomar decisões
  • Não utilizar o Serviço para fins ilícitos

6. Limitação de Responsabilidade

A Prazu se esforça para fornecer informações precisas, mas não garante a exatidão absoluta dos dados obtidos de fontes públicas (DJEN, DataJud). O usuário deve sempre conferir os prazos diretamente nos sistemas dos tribunais. A Prazu não se responsabiliza por perdas decorrentes de prazos incorretos ou notificações não entregues.

7. Privacidade

O tratamento de dados pessoais é regido pela nossa Política de Privacidade.

8. Alterações

Podemos alterar estes Termos a qualquer momento, notificando os usuários por e-mail ou pelo próprio Serviço. O uso continuado após alterações implica aceitação dos novos termos.
